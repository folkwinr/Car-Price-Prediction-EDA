## 2) Group-Based Filling (Key Strategy)
| Term | Meaning (B1‚ÄìB2) |
|---|---|
| group | a set of rows that share the same category (example: same make_model) |
| group key | a column used to create groups (make_model, body_type, etc.) |
| groupby | pandas method to group rows by one or more columns |
| segment | a ‚Äúsimilar car group‚Äù (example: same model + body type) |
| group-based imputation | filling missing values using values from the same group |
| hierarchical filling | filling with a strong group first, then weaker groups (fallback) |
| fallback | a backup step if the first fill method cannot fill values |
| transform | apply a group result to each row (example: group mean for each row) |
| mode | the most frequent value in a group |
| median | the middle value (more stable if outliers exist) |
| mean | the average value (can be affected by outliers) |

## 3) Methods We Used (When and Why)
| Term | Meaning (B1‚ÄìB2) |
|---|---|
| mode fill | fill missing values with the most common value in the group |
| median fill | fill missing values with the group median (robust to outliers) |
| mean fill | fill missing values with the group mean (average) |
| ffill (forward fill) | fill missing using the previous value in the group |
| bfill (backward fill) | fill missing using the next value in the group |
| propagation | filling by spreading existing values inside a group (ffill+bfill) |
| rule-based fill | filling by a logical rule instead of stats (example: warranty Yes/No) |
| domain rule | a rule based on real-world knowledge (example: Electric CO2) |
